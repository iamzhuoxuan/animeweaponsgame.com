# 功能概览（2025-11-24）

## 已实现功能
- **首页 Hero 区**：展示「Anime Weapons」标题、副标题与游戏简介，右侧嵌入发布视频；提供「Play on Roblox」与「Get Free Codes」CTA。
- **Active Codes（首页）**：从统一数据源加载当前可用的 Anime Weapons codes，支持复制，描述同步到主页与 Codes 页面。
- **Codes 专页**：检索/过滤、激活/过期分组、复制功能，FAQ、Pro Tips 与 CTA，显示动态 “Last Updated” 日期。
- **游戏特性展示**：Game Features 模块展示特色亮点与关于介绍，包含游戏跳转 CTA。
- **全局导航/页脚**：导航链接（Features/Codes/Wiki/Tier List/Games/Play），页脚包含快捷链接、社区与游戏入口；隐藏锚点保证 Wiki/Tier List/Games 导航可跳转。
- **一致图标与样式**：全站统一 SVG 图标替代 emoji，保持视觉一致性；Hero 标题行距优化避免字形裁剪。
- **SEO 与元信息**：为 Home/Codes 提供独立 meta title/description，canonical，OG/Twitter 元数据，站点级 WebSite JSON-LD。
- **站点数据与资源**：统一 codes 数据源 (`data/codes.ts`)，favicon/manifest 配置，Google Analytics（G-65KM1T79ZB），Google 站点验证文件。
- **搜索引擎友好性**：提供 `robots.txt`、`sitemap.xml`，并在 Vercel 配置指定 Next.js 框架与构建命令。

## 配置与部署
- **Vercel 配置**：`vercel.json` 指定 framework=nextjs、buildCommand=`npm run build`、outputDirectory=`.next`；`.vercelignore` 排除 `self_doc/`。
- **构建检查**：`npm run lint`、`npm run build` 通过（Next.js 14.2.18）。
